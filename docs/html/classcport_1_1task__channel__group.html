Detailed Description

template<typename Key>
class cport::task_channel_group< Key >

Provides mechanism to associate a task_cannel with a unique key.

All task_channel objects use shared task_scheduler instance.

Parameters
KeyMust meet the requirements of the Key template parameter type used in the standart class unordered_map.
